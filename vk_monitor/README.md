# VK Monitor (Windows, Python)

Приложение для мониторинга изменений в VK-сообществах с локальным хранением истории (SQLite).

## Возможности
- Добавление сообщества по ссылке.
- Первый запуск создает baseline snapshot (без генерации ложных "новых" событий).
- Ручная проверка и автоматическая проверка по таймеру.
- Отслеживаемые разделы: посты, закрепы, фото, фотоальбомы, видео, видеоальбомы, клипы, товары, обсуждения, ссылки, настройки (описание/статус/аватар/контакты).
- Мягкая деградация: при недоступности раздела сохраняется маркер "Раздел недоступен через API".
- Экспорт изменений в JSON/CSV.

## Запуск
```bash
cd vk_monitor
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app/main.py
```

## Настройка
1. Вставьте VK access token в поле "Токен".
2. Вставьте ссылку на сообщество и нажмите "Добавить сообщество".
3. На первом подключении создается снимок текущего состояния.
4. Нажмите "Проверить сейчас" для запуска сравнения с последним snapshot.

## Сборка .exe
```bat
cd vk_monitor
build_exe.bat
```
Готовый исполняемый файл будет в `dist/VKMonitor.exe`.

## Примечания
- Токен и интервал проверки хранятся локально в SQLite (`vk_monitor.db`).
- Для отдельных разделов API может быть недоступен из-за прав/настроек сообщества/лимитов.
